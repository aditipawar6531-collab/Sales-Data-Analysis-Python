# 📊 Advanced Sales Data Analysis using Python

## 📌 Project Overview
This project analyzes sales data using Python to identify top products and regions.

## 🚀 Features
- Product-wise sales analysis
- Region-wise performance
- Top 3 products identification
- Data visualization using charts

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib

## ▶️ How to Run
pip install pandas matplotlib  
python sales_analysis.py
